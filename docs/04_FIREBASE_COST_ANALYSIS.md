## Firebase Services ที่โปรเจคนี้ใช้จริง

| Service | ใช้งาน? | หลักฐานจากโค้ด |
|---------|:-------:|----------------|
| **Firestore (Database)** | ✅ ใช้ | `mockDb.ts` — import จาก `firebase/firestore` → อ่าน/เขียน collections: `rmas`, `users`, `brands`, `distributors`, `settings`, `counters` |
| **Authentication** | ✅ ใช้ | `mockDb.ts` — import จาก `firebase/auth` → Email/Password login เท่านั้น |
| **Hosting** | ✅ ใช้ | `firebase.json` → deploy static site จากโฟลเดอร์ `dist` ไปที่ `secrmssystem.web.app` |
| **Cloud Storage** | ❌ ไม่ใช้ | ไม่มี import `firebase/storage` ในโค้ดใดเลย |
| **Cloud Functions** | ❌ ไม่ใช้ | ไม่มี import `firebase/functions`, ไม่มีโฟลเดอร์ `functions/` |
| **Realtime Database** | ❌ ไม่ใช้ | ใช้ Firestore เท่านั้น |
| **Analytics** | ❌ ไม่ใช้ | ไม่มี import `firebase/analytics` |

## 6. วิธีตั้ง Budget Alert ป้องกันค่าใช้จ่ายเซอร์ไพรส์

ถ้าวันไหนอัพเกรดเป็น Blaze Plan สามารถตั้ง Budget Alert ได้:

1. ไปที่ [Firebase Console](https://console.firebase.google.com) → เลือก Project
2. ไปที่ **Usage and billing** → **Details & settings**
3. กดสร้าง **Budget alert** ตั้งไว้ที่ $1 (≈ ฿34)
4. ระบบจะส่ง Email แจ้งเตือนอัตโนมัติเมื่อใกล้ถึงงบ
